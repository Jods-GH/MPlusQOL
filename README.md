### Overview

MPlusQOL is a bundle of Mythic plus Quality of life additions.


### Features

- Repair Reminder (Automatically reminds you to repair when entering a resting area)
- Member died bar (Shows a bar when a member in your group dies)


Open the options by typing /mplusqol

For Feature suggestions or Bug reports please use Github or join the Discord
